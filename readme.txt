##########################
First time setup
##########################

-Run Calibrate.ahk
-Goto Nox
-Press F1
-Follow directions on top of screen.
-Run FFBE-MM.ahk

##########################
Usage & Controls
##########################
Reads .path files for modular exploration paths.
Combat and fight counter only work with the window active.
Macro will pause if not active.

-Run FFBE-MM.ahk
-Navigate to exploration selection screen.
-Press F9 to select exploration path.
-Press F8 to begin.

##########################
Troubleshooting
##########################

If you run into further issues try these steps first:
-Run Calibrate.ahk
-Goto Nox
-Press F11
-Select Yes
-Press F1
-Follow directions on top of screen.
-Run FFBE-MM.ahk

Disable windows screensavers & power settings to turn off your monitor.
It can mess with combat detection and cause crashes.
Any popups from the task bar can cause issues if they overlap Nox.
